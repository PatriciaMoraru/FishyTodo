# 🐠 FishyTodo

A visually-driven task manager where every task becomes a fish swimming in your personal tank. Designed with ADHD in mind.

**Live demo:** [patriciamoraru.github.io/FishyTodo](https://patriciamoraru.github.io/FishyTodo)

---

## Running locally

```bash
npm install
npm run dev
```

## Deploying

```bash
npm run deploy
```
